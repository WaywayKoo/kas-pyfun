    Python Level - Beginner

    Learn Datatypes

    There are three data types in Python

    Types Description
    int Int, short for integer. Whole number, any number without decimal point.
    float Floating-point numbers with up to 17 decimal points.
    str Str, String, words in general. Written inside quote.
    boolean True or False. Often use as a switch.
    Tip

    Learn the basics, but don't be scared to explore.

    Coding is all about exploring

    Variables

    So first of all, what are variables?

    Variable is a memory location to store datas. Think of variable as a box, and you can put any thing inside the box. You can later on call certain data by using the variable you created. Variable is a great tool to make your code look more simple.

    #1 Basics

    Code 

    1
2
3
4
    a = 10
print(a)
b = "10"
print(b)
    Output 
    10
10

    Tip

    Good variable name is important while working in a team. Always get in a good habits of using good variable name.

    A good variable name needs to follow the following: * Clear and Concise * No special character * No conflict with Python fuctions(sum, print, True...)

    Function useage
    title() Capitalize first letter of all word
    capitalize() Capitalize first letter of first word
    upper() Capitalize all
    lower() Lower all
    center(arg) text locat center of arg(int)
    ljust(arg) text ajust to the left of arg(int)
    rjust(arg) text ajust to the right of arg(int)
    split(arg) split str to list by arg(str)
    rsplit(arg) split str to list by arg(str) from right
    join(arg) join list to String by arg
    count(arg) count how many times arg(String) appear in String
    zfill() 依指定字元寬度補0
    strip() 兩邊移除指定字元
    lstrip() 左邊移除指定字元
    rstrip() 右邊移除指定字元
    index() 尋找指定字串，回傳索引。找不到則回傳 ValueError
    rindex() 尋找指定字串，回傳索引 (從右先找)
    find() 尋找指定字串，回傳索引。找不到則回傳 -1
    rfind() 尋找指定字串，回傳索引 (從右先找)
    replace() 取代指定字串
    splitlines() 依換行字元切割字串
    startswith() 判斷是否以指定字串開頭
    endswith() 判斷是否以指定字串結尾
